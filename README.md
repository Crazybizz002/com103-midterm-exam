# com103-midterm-exam
WHAT THE CODE DOES: 
It is a simple Mobile Legends match tracker. It asks for your IGN and rank, shows the hero list, lets you enter 4 matches, calculates KDA and a performance tag, then displays wins, losses, win rate, and the best match.

HOW TO RUN:
On Android, use Pydroid 3, paste the code, then tap Run.
On PC, save the file as mlbb_tracker.py, then run python mlbb_tracker.py in the terminal.
